chimbalix-installer
============

Customizable GUI installer adapted for [Chimbalix Linux](https://github.com/Shedou/Chimbalix).

Supports "portsoft" directory.

### How To Build DEB Package:
Build-Depends: debhelper (>= 10) libzxcvbn-dev qtbase5-dev qttools5-dev-tools

Run in a terminal (inside the source folder): fakeroot ./debian/rules binary

### Settings:
Check folder: /usr/share/chimbalix-installer/data
